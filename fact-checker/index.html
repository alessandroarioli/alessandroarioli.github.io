API Keys Setup

This tool uses two free APIs that require a key. Both are free to obtain:

1. Google Fact Check Tools API — 1,000 queries/day free
→ console.cloud.google.com → Create project → Enable "Fact Check Tools API" → Credentials → Create API Key

⚠ No key set

2. The Guardian Open Platform — 5,000 queries/day free
→ Register at open-platform.theguardian.com/access → Choose "Developer" tier → receive key by email in minutes

⚠ No key set — Guardian results may be rate-limited
